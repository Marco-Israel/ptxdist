\chapter{Final images}%
\label{cha:Final images}


\section{Compleate system imagese}%
\label{sec:compleate system images}}

Complete system images:

\begin{itemize}
    \item sd.hdimg : is a bootable image primarily for SD Card, but works for eMMC, too.
    \item boot-mlo.vfat : fat filesystem with MLO, u-boot and linux-kernel
        \tem root.ext2 : ext2 rootfs
\end{itemize}


\section{Bootloader images}%
\label{sec:Bootloader images}

Bootloader images:

\begin{itemize}
    \item u-boot.bin
    \item u-boot.img : u-boot image for SD Card/eMMC
    \item u-boot_spi.bin
    \item u-boot_spi.img : u-boot image for SPI NOR Flash
    \item MLO : (second program loader SPL) for SD Card/eMMC
    \item MLO_spi.byteswap
\end{itemize}


\section{Liunx images}%
\label{sec:Liunx images}

Linux images:

\begin{itemize}
    \item linuximage : Kernel (uImage)
    \item am335x-mba335x.dtb : device tree blob for TQMa335x on MBa335x
\end{itemize}

\section{Root fs imges}%

Rootfs images:
